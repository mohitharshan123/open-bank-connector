# Bank Demo App

React demo application using TanStack Query to interact with the Banking API Gateway.

## Setup

```bash
pnpm install
pnpm dev
```

App runs at `http://localhost:5173`

## Prerequisites

- Bank Microservice (port 3000)
- API Gateway (port 3001)
- MongoDB
- Redis

## Features

- Authenticate with banking providers
- View account details
- View balances by currency

## Tech Stack

- React 18 + TypeScript
- TanStack Query
- Tailwind CSS
- Vite
